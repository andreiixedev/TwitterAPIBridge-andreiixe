What is this?

A server that translates the Bluesky API into the Twitter API v1.0 format. This allows you to use old Twitter clients with modern Bluesky, giving vintage devices new life without custom apps.

Frequently Asked Questions

What Twitter clients are compatible? ‚ñº

Tested with clients from 2010-2012 era. See compatibility list ‚Üí

Why Bluesky instead of actual Twitter/X? ‚ñº
  1. Twitter API costs $5,000+/month
  2. Extreme hostility to third-party clients
  3. Authentication would be near impossible
  4. Bluesky is open and welcoming to developers
Can I use mobile app API keys? ‚ñº

No. This violates Twitter's TOS, would get users banned, and keys have restricted access with no documentation.

Support other platforms like Mastodon? ‚ñº

Mastodon is planned for the future. Other platforms are not currently planned.

Twitter API v1.1 support? ‚ñº

Planned! But v1.0 support needs to be completed first.
